## Мои конфиги (Dotfiles)
Управление конфигурационными файлами с помощью GNU Stow.
## Структура

- .config/ — пользовательские конфиги (разворачиваются в $HOME/.config).
- keyd-package/ — системный конфиг для keyd (разворачивается в /etc/default/).
## Установка
1. Подготовка
Убедитесь, что stow установлен:
bash

```bash
sudo apt install stow  # Ubuntu/Debian
sudo pacman -S stow    # Arch
```

2. Развертывание пользовательских конфигов
Для файлов, которые должны попасть в домашнюю директорию (~/.config и т.д.):
bash

# Из корня репозитория dotfiles
```bash
stow .config
```

3. Развертывание системного конфига (keyd)

## Так как keyd требует прав root и привязки к корню системы /:

```bach
sudo stow -t / keyd-package
```

Используйте код с осторожностью.
Откат изменений (удаление ссылок)
Если нужно безопасно удалить симлинки:

```bash
stow -D .config
sudo stow -D -t / keyd-package
```

Используйте код с осторожностью.
